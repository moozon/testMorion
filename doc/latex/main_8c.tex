\doxysubsubsection{\texorpdfstring{BAUDRATE}{BAUDRATE}}
{\footnotesize\ttfamily \#define BAUDRATE~115200}



Скорость передачи данных USART. 

Тестовое задание\+: test\+Morion. \begin{DoxyVersion}{Версия}
1.\+0.\+0
\end{DoxyVersion}
Описание\+:
\begin{DoxyItemize}
\item МК\+: STM32\+F103\+C8
\item Светодиод подключен к пину\+: PB6
\item USRAT1\+: TX=PA9, RX=PA10
\item Ввод/вывод интерфейса USART реализован через канал DMA
\end{DoxyItemize}

Работа программы\+: При первоначальном запуске программы светодиод мигает в режиме 500мс включен, 500мс выключен. Можно изменять режим свечения светодиода нижеописанными командами.

Реализовано\+:
\begin{DoxyEnumerate}
\item Команда \char`\"{}\+Blink X(ms) Y(ms)\char`\"{} -\/ Задает режим свечения светодиода с заданным временем включения/выключения(X/Y).
\item Команда \char`\"{}\+Fade X(ms)\char`\"{} -\/ Задает режим плавного зажигания и затухания светодиода, используя Шим на Timer4.
\item Команда \char`\"{}\+Stop\char`\"{} -\/ Вводит МК в режим зависания и через 10 сек. срабатывает таймер Watchdog(\+IWDG) и МК перезагружается.
\item Приглашение пользователк в консоле к выполнению команд с примерами.
\item Оформлены комментарии к коду в формате Doxygen и сгенерирована соответствующая документация.
\item Создан git репозиторий и выгружен в удаленный репозиторий на Git\+Hub(\href{https://github.com/moozon/testMorion}{\texttt{ https\+://github.\+com/moozon/test\+Morion}}). 
\end{DoxyEnumerate}\mbox{\Hypertarget{main_8c_af281425e62298bac2df0fbe8690a4844}\label{main_8c_af281425e62298bac2df0fbe8690a4844}}
